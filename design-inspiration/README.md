# Design Inspiration

Denna mapp innehåller kod och designreferenser från V0-templates och andra designkällor som används som inspiration för CrowdVine-projektet.

## Struktur

Varje designreferens ska ligga i sin egen undermapp med följande struktur:

```
[projekt-namn]/
├── components/     # React-komponenter
├── app/           # Next.js app-struktur (om tillämpligt)
├── styles/        # CSS/Tailwind-stilar
└── README.md      # Beskrivning av projektet
```

## Lägga till ny designreferens

1. Skapa en ny mapp med projektets namn
2. Kopiera relevanta filer från V0 eller annan källa
3. Lägg till en README.md med:
   - Länk till originalkällan
   - Beskrivning av designelement
   - Noteringar om vad som är intressant för CrowdVine

## Användning

När du lägger till kod här kan AI-assistenten:
- Analysera designprinciperna
- Identifiera återanvändbara patterns
- Anpassa dem till CrowdVine-projektet
- Integrera dem i befintliga komponenter
