### Auth gallery images

Put the images you want to use for:
- `/access-request`
- `/log-in`

in this folder.

#### Recommended
- Put them in `PACT_ACCESS_2/` and name them:
  - `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`, `5.jpg`, `6.jpg`, `7.jpg`, `8.jpg`
- Format: **jpg** or **webp** (webp is best performance, but jpg works)
- Orientation: **portrait or square** works best for the 3D gallery

After adding/replacing images, reload the page.
