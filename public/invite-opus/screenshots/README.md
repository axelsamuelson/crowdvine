# Platform screenshots for invite landing page

Lägg dina screenshots här för att använda dem i "This is how it works"-sektionen.

## Namnkonvention

För att bilderna ska användas automatiskt, döp dem enligt följande:

| Fil | Beskrivning |
|-----|-------------|
| `01-discover.jpg` | Shop-sida – bläddra viner från producenter |
| `02-reserve.jpg` | Varukorg eller checkout – reservera flaskor |
| `03-pallet.jpg` | Pallet-sida – se hur palleten fylls |
| `04-delivery.jpg` | Leverans eller bekräftelse |

Alternativt kan du använda numrerade namn: `1.jpg`, `2.jpg`, `3.jpg`, `4.jpg`.

## Format

- JPG eller PNG
- Rekommenderad storlek: minst 600×400 px (3:2 aspect ratio fungerar bra)
- Bilderna visas i karusellen och i "how it works"-korten
